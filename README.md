# Customer Churn Analysis Dashboard (Tableau)

## Project Overview
This project analyzes customer churn patterns using Tableau to identify the key factors that contribute to customer attrition. The interactive dashboard allows users to explore churn trends across different customer segments and service characteristics.

Understanding why customers leave helps organizations develop targeted retention strategies and improve long-term customer value.

## Objectives
- Identify customer segments with high churn rates
- Analyze churn by contract type, tenure, and service usage
- Visualize patterns that indicate potential churn risk
- Provide an interactive dashboard for exploring retention patterns

## Tools & Technologies
- Tableau - dashboard development
- Data visualization - exploration of churn patterns and segmentation
- Business analytics - interpretation of churn drivers and retention insights

## Dashboard Features
The dashboard enables users to:

- Explore churn rates across multiple customer attributes
- Filter results by contract type, tenure, and service features
- Identify high-risk customer groups
- Examine patterns that may inform retention strategies
- Interactive filters allow analysts and business stakeholders to quickly investigate different customer segments and uncover trends in churn behavior.
