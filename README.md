# Customer Churn Analysis Dashboard (Tableau)

This project analyzes customer churn patterns using Tableau to identify key drivers of customer attrition.

## Project Goals
- Identify segments with high churn rates
- Analyze churn by contract type, tenure, and service usage
- Provide an interactive dashboard for exploring retention patterns

## Tools Used
- Tableau
- Data visualization
- Business analytics

## Files
- Customer Churn Dashboard_D601.twbx – Tableau packaged workbook containing the dashboard and dataset
- Customer Churn Dashboard Instructional.docx – project explanation and instructions

## Overview
The dashboard provides interactive filters allowing users to explore churn patterns across customer segments and identify potential retention strategies.
